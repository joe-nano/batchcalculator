Features

Batch calcualtor allows you to:

  • easily calculate the correct amount of chemicals (batch) from a composition given by the molar ratios.
  • generate a nice, cutomizable pdf reports (or possibly other formats)
  • store, reuse and share a well structured database of chemicals, and components to save time
  • interactively explore the batch composition by changing the molar ratios, chemicals used, concentrations etc.
  • multiplatform, written in python so any platform is supported
