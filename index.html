Features

Batch calcualtor allows you to:

  • easily calculate the correct amount of chemicals (batch) from a composition given by the molar ratios.
  • generate a nice, cutomizable pdf reports (or possibly other formats)
  • store, reuse and share a well structured database of chemicals, and components to save time
  • interactively explore the batch composition by changing the molar ratios, chemicals used, concentrations etc.
  • multiplatform, written in python so any platform is supported

Download

TeX and pdf

In order to be able to get pdfs and not only tex files of the reports you need to have a TeX distibution installed. For general information about TeX youcan visit TUG webpage. Major TeX packages for different operating systems are:

  • TeX Live for Unix/Linux
  • MacTeX for Mac OS
  • MikTeX for Windows

Editing the database

The program uses a customizable database of chemicals that can be edited using external programs such as sqlitebrowser.
