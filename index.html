--- title: Batch Calculator layout: default ---

Features

  • simplify the calculation of the batch composition and reactant amounts
  • allow interactive explorations by changing the composition, reactants, sample size etc.
  • get a customizable lab report (pdf and/or TeX)
  • store all the syntheses in a database for bookkeeping and sharing with others
  • build, edit, reuse and share a well structured database of chemicals, and components to save time
  • multiplatform, written in python which makes coding and adding your own features easy
  • fast, easy, reproducible, collaborative, extendable and free!
