Features

Batch calcualtor allows you to:

  • easily calculate the correct amount of chemicals (batch) from a composition given by the molar ratios of the components
  • generate nice, cutomizable pdf reports (from TeX or directly)
  • build, edit, reuse and share a well structured database of chemicals, and components to save time
  • interactively explore the batch composition by changing the molar ratios, chemicals used, concentrations etc.
  • multiplatform
  • written in python which makes coding and adding your own features easy
