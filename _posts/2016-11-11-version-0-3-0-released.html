Batch Calculator v0.2.0

New features:

  • New internal methods for adding/editing/deleting database entries (no external software required anymore!).
  • Option to create an new database from scratch.
  • New option to generate pdf reports directly (TeX is still an option but if you don't like it now there is an alternative).
  • pdf reports can be generated not only from batch calculation (moles to masses) but also from the composition calculation (masses to moles).
  • Program can now handle chemicals that are multicomponent mixtures.

Other improvements

  • Volumes are now included next to masses in the results panel (for chemicals that are liquids and have their densities in the database).
  • Additional scaling of the results is now possible.
  • Several annoying bugs were corrected.
